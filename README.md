# Intro to R Tutorials 

This repository contains [*learnr*](https://rstudio.github.io/learnr/) tutorials introducing R and basic stats concepts through exercises and MC quizzes. They were used to introduce medical students to R in *Modul 28: Einblicke in die Praxis der empirischen neurowissenschaftlich-psychiatrischen Forschung* (Charité - Universitätsmedizin Berlin, DE) and last updated in the summer term of 2023.

The **Intro_to_R** tutorial introduces basic R concepts (environment, variables, data frames, libraries etc), as well as Pearson's correlation and the independent t-test. This tutorial is the basis of the course and, like all others, can be completely self-driven. 

The data used (CSV files) are simulated questionnaire scores and behavioral measures.
